Using Nano

The nano command in Linux is a basic terminal text editor used to create and edit text files. Nano is simple to use and beginner-friendly compared to other editors like Vim.

Examples

nano myfile.txt - This command opens or creates a file called myfile.txt for editing in Nano.

Common commands include:

  • Ctrl+X: Exit the editor
  • Ctrl+O: Save the file
  • Ctrl+W: Search for text

Scenario 1: Editing a Configuration File

Task: You need to modify a file in /tmp called config.txt.

Command: nano /tmp/config.txt

Next Step: After editing, use Ctrl+O to save changes and Ctrl+X to exit Nano.
