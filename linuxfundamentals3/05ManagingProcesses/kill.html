Using the kill Command

The kill command is used to terminate processes in Linux. It sends signals to processes to request their termination, with different signals offering different ways to stop a process.

Common Signals:

SIGTERM (15): Gracefully stops a process, allowing it to clean up before exiting.

SIGKILL (9): Forcibly stops a process without cleanup.

SIGSTOP: Pauses a process without terminating it.

Example:

kill -9 1337 - This command forces the termination of the process with PID 1337.

Scenario 8: Gracefully Stop a Script

Task: Stop a script running with PID 2025 without forcing it.

Command: kill -15 2025

Next Step: If the process does not terminate, escalate to kill -9 2025.
