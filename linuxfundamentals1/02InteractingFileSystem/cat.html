Understanding the cat Command

The cat command (short for "concatenate") is used to display the contents of files in Linux.

Usage:

cat is commonly used to view text files, system logs, and combine multiple files into one.

Examples:

  • cat report.txt - Displays the contents of the file report.txt.
  • cat /var/log/syslog - Outputs the contents of the system log.

Scenario 3: Viewing Log Files

Task: After navigating to the system_logs folder, you want to view the contents of the file log1.txt.

Command: cat /tmp/system_logs/log1.txt

Example Output:

  • Error: Disk space running low on /dev/sda1
  • Warning: High memory usage detected
