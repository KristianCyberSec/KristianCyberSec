Understanding the pwd Command

The pwd command (print working directory) is used to display the full path of the current directory.

Usage:

pwd is used to find your current location within the file system.

Example:

pwd might return /home/user/Documents if you're inside the "Documents" folder.

Scenario 4: Finding Your Current Directory

Task: After navigating through several directories, you want to confirm where you are.

Command: pwd

Example Output: /tmp/system_logs
