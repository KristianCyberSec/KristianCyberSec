Interacting with the Filesystem

ls (list): Lists the contents of a directory.

Usage: ls is used to quickly check the contents of directories.

Examples:

  • ls lists the contents of the current directory.
  • ls -l lists the contents with detailed information like file size, permissions, and dates.
  • ls -a shows hidden files (those starting with a dot .).

Scenario 1: Searching and Listing Files

Task: You want to search for files or directories with the name "Logs" inside a specific folder like /tmp.

Command: ls /tmp | grep "Logs"

Explanation: ls /tmp lists all files and directories in /tmp, and piping (|) the output to grep filters for entries containing "Logs."

Example Output:

  • system_logs
  • backup_logs

Next Step: Now that you've located the folder system_logs, you want to list its contents.

Command: ls /tmp/system_logs

Example Output:

  • log1.txt
  • log2.txt
