Using the mkdir Command

The mkdir command in Linux is used to create new directories. It’s a simple but powerful command for organizing files into folders for better structure and management.

Examples

mkdir backup - This command creates a new directory called backup.

Scenario 5: Creating a Folder

Task: You want to create a folder called documents in the /home/user directory.

Command: mkdir /home/user/documents

Next Step: Use ls /home/user to verify the folder has been created.
