Using the su Command

The su command in Linux is used to switch to another user account. This is useful when you need to perform tasks with different user privileges.

Examples

su root - This command switches the current session to the root user account, prompting you for the root user's password.

Scenario 10: Switching Users

Task: You want to switch to user2 to perform tasks with their permissions.

Command: su user2

Next Step: Enter the password for user2 and verify the switch using the whoami command.
