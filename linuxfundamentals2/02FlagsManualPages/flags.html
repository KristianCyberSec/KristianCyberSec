Using Flags with Commands

Flags (also known as switches) modify command behavior. They are typically prefixed by - or -- to change how a command behaves or what output it provides.

Usage:

ls -a lists all files, including hidden ones.

Common Flags:

  • -a: Show hidden files.
  • -l: Show detailed information about files.
  • --help: Display help documentation for a command.

Scenario 2: Using Flags with Commands

Task: You want to list all files, including hidden ones, in the /var directory.

Command: ls -a /var

Next Step: Use ls --help to learn about additional flags, such as -l for detailed views or -h for human-readable sizes.
