Linux Fundamentals 2 - Commands with Real-World Scenarios

In Linux Fundamentals 2, the focus shifts toward using SSH for remote access, learning how to use flags, creating/moving/deleting files, switching users, and understanding important system directories.

SSH (ssh): Connect to remote Linux systems securely.

Flags: Modify command behavior (e.g., ls -a to show hidden files).

Manual Pages (man): Learn more about commands with man <command>.

File Creation and Management (touch, mkdir, rm, cp, mv, file): Commands for creating, copying, moving, and removing files and directories.

Permissions (chmod, chown): Adjust file permissions and ownership.

Switching Users (su, sudo): Switch between users or run commands with elevated privileges.

Important Directories:

/etc: System configuration.

/var: Variable data like logs.

/tmp: Temporary files.

/root: Root user's home directory.
